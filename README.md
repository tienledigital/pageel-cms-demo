# Pageel CMS Demo Content

This repository is designed to demonstrate and test the capabilities of **Pageel CMS**.

## Features Tested

- [x] Multi-collection support (Blog & Docs)
- [x] Custom frontmatter templates
- [x] Independent content paths
- [x] Git-based configuration (`.pageelrc.json`)

## Structure

- `src/content/blog/`: Blog posts with Astro layout compatibility.
- `src/content/docs/`: Documentation files.
- `public/images/`: Assets.

## Setup for Pageel CMS

1. Point Pageel CMS to this repository.
2. Use your Personal Access Token (PAT).
3. The `.pageelrc.json` will be automatically detected.
